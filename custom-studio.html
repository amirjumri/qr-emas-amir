2) Add Layer
Hotkey:
• Delete = padam layer
• Ctrl+D = duplicate layer
• Shift+Click = multi select (pilih banyak layer)
• Ctrl+G = group, Ctrl+Shift+G = ungroup
• Ctrl+Z = undo, Ctrl+Y / Ctrl+Shift+Z = redo
• Double-click gambar = crop mode (Enter siap, Esc batal)
• Drag layer dalam senarai = susun atas/bawah
