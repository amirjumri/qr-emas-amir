Split Walk-In (J916)

Khas untuk WALK-IN — order PENDING dan tiada Bill ID online. Tiada refund. Tiada pulang duit.
Mode: PENDING → PAID (bayar manual selepas simpan)
Parameter URL: ?order=XXXXXXX
Kalau ada Bill ID online → ini checkout online, split walk-in akan block.
Ini tak akan berubah. Simpan dalam weight_g dan/atau weight_orig_g.
Ini disimpan dalam weight_new_g (tak override weight_g).
Ini akan jadi nilai “Perlu bayar” dalam Admin-J916.
Checkout group: -
Apa yang akan berlaku bila “Simpan Split Walk-In”:
  • Status kekal PENDING
  • Harga order akan jadi Harga baru (update amount_cents + subtotal_rm + grand_total_rm)
  • Berat asal kekal (tidak override weight_g)
  • Berat baru disimpan dalam weight_new_g (dan lock asal dalam weight_orig_g jika belum ada)
  • Selepas itu, Amir balik Admin-J916 → klik Bayar manual
Tip: Lepas simpan, buka semula Admin-J916 dan tekan Refresh. Nilai “Perlu bayar” akan ikut harga baru.
